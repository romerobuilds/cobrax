Depois que você criar ou editar arquivos no VS Code, no Git Bash faça:

git add . && git commit -m "..." && git push

Depois no servidor ssh cobrax@95.216.138.163:

git pull origin main
docker compose down
docker compose up -d --build

Forçar:

cd ~/apps/cobrax
git status
git log -1 --oneline

git fetch origin
git reset --hard origin/main


Rebuildar Docker:

docker compose down
docker compose up -d --build
